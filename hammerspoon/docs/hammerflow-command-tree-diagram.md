## Command Depth Analysis

### Depth 1 (Immediate)
- `F18 вҶ’ t` - Launch WezTerm
- `F18 вҶ’ c` - Launch Cursor
- `F18 вҶ’ s` - Show Dashboard

### Depth 2 (One Group)
- `F18 вҶ’ h вҶ’ r` - Reload Hammerspoon
- `F18 вҶ’ w вҶ’ h` - Window Left Half
- `F18 вҶ’ a вҶ’ p` - Toggle TV Power
- `F18 вҶ’ l вҶ’ p` - LG Monitor Power On

### Depth 3 (Two Groups)
- `F18 вҶ’ a вҶ’ v вҶ’ u` - HA Volume Up
- `F18 вҶ’ a вҶ’ i вҶ’ 1` - HA Switch HDMI 1
- `F18 вҶ’ l вҶ’ i вҶ’ 2` - LG Switch HDMI 2

### Depth 4 (Three Groups)
- None currently (but structure supports it)

## Key Frequency Map

### Most Used (Top Level)
- `t` - Terminal (WezTerm)
- `c` - Code Editor (Cursor)
- `s` - Status Dashboard

### Common Groups
- `a` - Home Assistant (most complex)
- `w` - Window Management
- `l` - LG Monitor

### Specialized Groups
- `h` - Hammerspoon config
- `y` - System utilities
- `p` - Prefixes

## Command Patterns

### Volume Control Pattern
```
[a] вҶ’ [v] вҶ’ u/d  (HA Volume)
[l] вҶ’ u/d        (LG Volume)
```

### Input Switching Pattern
```
[a] вҶ’ [i] вҶ’ 1/2/3/4  (HA HDMI)
[l] вҶ’ [i] вҶ’ 1/2/3/4  (LG HDMI)
```

### Power Control Pattern
```
[a] вҶ’ p  (HA TV Power)
[l] вҶ’ p/o  (LG Power On/Off)
```

### Navigation Pattern
```
[a] вҶ’ [i] вҶ’ h  (Home)
[a] вҶ’ [i] вҶ’ b  (Back)
```

## Memory Aids

### Mnemonics
- **t** = **T**erminal
- **c** = **C**ursor/Code
- **b** = **B**rowser
- **f** = **F**inder
- **s** = **S**tatus
- **h** = **H**ammerspoon
- **w** = **W**indow
- **a** = **A**ssistant (Home Assistant)
- **l** = **L**G Monitor
- **y** = s**Y**stem
- **p** = **P**refixes

### Patterns
- **u/d** = Up/Down (volume, navigation)
- **p/o** = Power On/Off
- **1/2/3/4** = HDMI inputs
- **h/b** = Home/Back (navigation)
