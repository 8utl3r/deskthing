# Agent Documentation Compliance

## Compliance Verification Process

### Pre-Action Checklist
Before any significant action:
- [ ] Read `project_context.md` fully
- [ ] Identify applicable rules
- [ ] Complete mandatory checklist items
- [ ] Document planned action
- [ ] Verify rule compliance

### Session Documentation Requirements
- [ ] Record session start with objective and next steps
- [ ] Document key decisions as they're made
- [ ] Update actions taken after each significant change
- [ ] Record session end with summary and next steps
- [ ] Commit documentation changes

### Rule Adherence Monitoring
- [ ] Check for rule violations after each action
- [ ] Document any violations immediately
- [ ] Analyze root cause of violations
- [ ] Implement prevention strategies
- [ ] Update compliance processes

## Violation Response Protocol

### When Violation Detected
1. **STOP** all current actions
2. **DOCUMENT** violation in `rule_violation_tracking.md`
3. **ANALYZE** why rule wasn't followed
4. **CORRECT** action to follow rule
5. **RESUME** only after full compliance

### Violation Documentation Format
- **Date/Time**: When violation occurred
- **Rule Violated**: Specific rule that was broken
- **Action Taken**: What was done instead
- **Root Cause**: Why rule wasn't followed
- **Prevention Strategy**: How to prevent future violations

## Compliance Metrics
- **Zero Violations**: Target for each session
- **Complete Documentation**: All actions recorded
- **Proper Integration**: Dotfiles updated appropriately
- **Process Adherence**: All rules followed consistently

## Enforcement Mechanisms
- **Immediate Stop**: Halt on violation detection
- **Mandatory Documentation**: Record all violations
- **Root Cause Analysis**: Understand why violations occur
- **Prevention Strategies**: Implement improvements
- **Regular Review**: Assess compliance patterns

## Success Indicators
- All rules followed without exception
- Complete session documentation
- Proper dotfiles integration
- Consistent process adherence
- Zero rule violations per session
