# Pre-Action Checklist

## Version History
- 2026-01-20: Initial checklist.
- 2026-02-23: Removed hard line-count checklist requirement.

## Mandatory Pre-Action Requirements
- [ ] Identify applicable rules
- [ ] Read relevant rule files completely
- [ ] Read `project_context.md` fully
- [ ] Document planned action in session records
- [ ] Verify rule compliance before proceeding

## Before Installing/Configuring Tools
- [ ] Check `/Users/pete/dotfiles` for existing configuration
- [ ] Plan dotfiles update strategy
- [ ] Document installation in session records

## Before Code Changes
- [ ] Read `project_context.md` fully
- [ ] Document planned changes in session records
- [ ] Check document organization/readability; split only if it improves clarity

## Before File Creation
- [ ] Check for existing similar files
- [ ] Verify naming convention (lowercase with underscores)
- [ ] Plan file organization (docs/ subfolder if appropriate)

## Before Session End
- [ ] Update `project_context.md` with session summary
- [ ] Record next 3 specific steps
- [ ] Note blockers or concerns
