# Home Assistant Integration

Scripts for managing and integrating with Home Assistant automation system.

## Scripts

- **`ha-analyze`** - Analyze Home Assistant configuration
- **`ha-diagnose`** - Diagnose Home Assistant issues
- **`ha-explore`** - Explore Home Assistant entities and services
- **`ha-explore-simple`** - Simplified exploration tool
- **`ha-export-guide`** - Export configuration guides
- **`ha-get-token`** - Get authentication tokens
- **`ha-pull`** - Pull configuration from remote server
- **`ha-pull-api`** - API-based configuration pulling
- **`ha-pull-api-v2`** - Updated API pulling
- **`ha-sync`** - Sync local changes to remote server
- **`ha-test-integration`** - Test Home Assistant integrations
- **`ha-tv-volume`** - Control TV volume via Home Assistant
- **`ha-tv-wol`** - Wake-on-LAN for TV control
- **`ha-validate`** - Validate YAML configurations
