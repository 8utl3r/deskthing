# SK1Transfer USB – Contents and Usage

This USB drive contains firmware and scripts for the Ugoos SK1 flash process and Windows PC inventory.

---

## Contents

| File | Purpose |
|------|---------|
| `SK1_2.0.5.img` | Android 14 firmware image for Ugoos SK1 (~1.5 GB) |
| `windows-system-inventory.ps1` | PowerShell script to gather Windows system info |
| `inventory-rich.py` | Python script with Rich dashboard (installs rich if needed) |
| `run-from-usb.ps1` | Finds USB, runs inventory, saves TOC (for SSH from Mac) |
| `RUN_INVENTORY.bat` | Double-click to run inventory with Rich UI; saves to TOC.md |
| `TOC.md` | Table of contents (generated by running inventory) |
| `README.md` | This file |

---

## Firmware Flash (SK1)

1. On the Windows PC: **Setting → Load Img** in AML Burning Tool.
2. Select `SK1_2.0.5.img` from this USB drive.
3. Click **Start** in the tool.
4. Put the SK1 in burn mode: hold Recovery (toothpick in 3.5 mm jack) → connect power → wait ~5 s.
5. Wait for **Burn Success**.

## System Inventory (Table of Contents)

Generates a markdown report with OS, hardware, drivers, firewall, installed apps, services, network, and more.

### On Windows (run locally)

1. Double-click `RUN_INVENTORY.bat`, or
2. Open PowerShell in this folder and run:
   ```powershell
   .\windows-system-inventory.ps1 | Out-File -FilePath TOC.md
   ```
3. Open `TOC.md` to view the report.

### From Mac (via SSH)

Plug USB into Windows PC first. Then from Mac:

```bash
~/dotfiles/scripts/ugoos/run-inventory.sh
```

Or save TOC to a local path:

```bash
~/dotfiles/scripts/ugoos/run-inventory.sh ~/Downloads/windows-TOC.md
```

Uses Rich progress dashboard while running.

## What the Inventory Collects

- **OS:** Name, version, build, architecture
- **Hardware:** CPU, RAM, disks, BIOS
- **Network:** Adapters, IP config
- **Firewall:** Profiles, rules
- **Drivers:** Third-party (run as Admin for full list)
- **Software:** Installed apps from registry
- **Services:** Auto-start services
- **Startup:** Startup programs
- **Updates:** Recent hotfixes
- **.NET:** Installed versions
- **Security:** Antivirus products
- **USB:** Devices (including Amlogic for SK1)
