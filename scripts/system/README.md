# System Management Scripts

Core dotfiles management and system setup scripts.

## Scripts

- **`bootstrap`** - Initial system setup and configuration
- **`link`** - Create symlinks from dotfiles to system locations (with dry-run and backup)
- **`snapshot`** - Create system inventory and package lists
- **`update`** - Update system packages and configurations

## Usage

These scripts form the core of the dotfiles management system. Run `./bootstrap` for initial setup, then use `./link` to apply configurations.
