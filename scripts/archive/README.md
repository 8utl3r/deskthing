# Archive Directory

This directory contains test files, development scripts, and experimental code that are no longer needed in the main bin directory but are preserved for reference.

## Contents

### LG C5 Development Files
- **`lg-test-*`** - Various test scripts for LG C5 monitor control development
- **`lg-debug`** - Debug tools for LG monitor troubleshooting
- **`lg-server*`** - Server and connection testing scripts
- **`lg-start`**, **`lg-terminal`** - Development utilities
- **`investigate-9761`** - Port investigation script
- **`lg_tv (2025).xml`** - Savant Blueprint profile for LG TVs

## Purpose

These files document the development process and testing that led to the final working solutions in the main directories. They are kept for:
- Historical reference
- Debugging future issues
- Understanding the development process
- Potential future development

## Note

These files are not part of the active dotfiles system and should not be used for production purposes.
